# 우아한테크코스 프리코스 2주차

## 자동차 경주 기능목록

### 자동차 이름을 입력하는 기능
[] 자동차 이름 입력하기
[] 입력된 결과를 쉼표 기준으로 구분하기

### 시도할 횟수를 입력하는 기능
[] 시도할 횟수 입력하기
[] 입력한 횟수만큼 게임 진행

### 자동차가 전진하는 기능
[] 횟수가 증가할 때마다 각 자동차가 0 ~ 9 사이의 랜덤값 뽑기
[] 뽑은 랜덤값이 4 이상일 경우 해당 자동차 전진(’-’를 1개 추가)

### 우승자를 정하는 기능
[] 각 자동차의 최종 ‘-’의 개수 구하기
[] ‘-’의 개수가 가장 많은 자동차 판단하기
[] 최종 우승자 출력하기

### 예외 처리 기능
[] 자동차 이름이 5자를 초과할 경우
[] 시도 횟수가 문자열인 경우
